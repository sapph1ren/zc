# Zipcord

## About

Fast, secure and optimized messenger as an ark for Slavs users. Everything in C17, ~10Mb in RAM, Zcrypt system, zero http/-s. 

## libs?
- Nuklear 
- GDI (only for Windows)
- libuv
- miniaudio 
- sqlite3
